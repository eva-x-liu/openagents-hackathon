# 中国居民膳食指南（2022）核心规则

> **来源**：中国营养学会《中国居民膳食指南（2022）》  
> **用途**：约束 AI 输出，确保饮食建议符合权威指南  
> **版本**：v2.1（基于中国营养学会官方资料更新）

---

## 一、平衡膳食八准则

### 准则一：食物多样，合理搭配
- 每天摄入 **12种以上** 食物，每周 **25种以上**
- 膳食应包括：谷薯类、蔬菜水果类、畜禽鱼蛋奶类、大豆坚果类
- 谷类为主，每天摄入谷类食物200-300g，其中全谷物和杂豆类50-150g

### 准则二：吃动平衡，健康体重
- 各年龄段人群都应天天进行身体活动
- 保持能量平衡，维持健康体重
- 食不过量，保持能量平衡
- 成年人每周至少进行5天中等强度身体活动，累计 **150分钟以上**
- 主动身体活动最好每天 **6000步**
- 鼓励适当进行高强度有氧运动，加强抗阻运动，每周2-3天
- 减少久坐时间，每小时起来动一动

### 准则三：多吃蔬果、奶类、全谷、大豆
- 蔬菜水果、全谷物和奶制品是平衡膳食的重要组成部分
- 餐餐有蔬菜，保证每天摄入 **300-500g** 蔬菜，深色蔬菜应占1/2
- 天天吃水果，保证每天摄入 **200-350g** 新鲜水果
- ⚠️ **果汁不能代替鲜果**

### 准则四：适量吃鱼、禽、蛋、瘦肉
- 鱼、禽、蛋类和瘦肉摄入要适量，平均每天 **120-200g**
- 优先选择鱼和禽类
- 蛋类摄入 **40-50g/天**，相当于每天1个鸡蛋
- ⚠️ **吃鸡蛋不弃蛋黄**
- 少吃加工肉制品

### 准则五：少盐少油，控糖限酒
- 成人每天食盐摄入 **不超过5g**（2022版从6g下调至5g）
- 烹调油 **25-30g**
- 添加糖摄入 **不超过50g**，最好控制在25g以下
- 反式脂肪酸每天摄入不超过2g
- 不喝或少喝含糖饮料
- 儿童青少年、孕妇、乳母以及慢性病患者不应饮酒
- 成年人如饮酒，一天酒精量不超过15g

### 准则六：规律进餐，足量饮水
- 合理安排一日三餐，定时定量，不漏餐，每天吃早餐
- 早餐能量占全天25-30%，午餐30-40%，晚餐30-35%
- 规律进餐、饮食适度，不暴饮暴食、不偏食挑食、不过度节食
- 足量饮水，少量多次
- 成年男性每天喝水 **1700ml**，成年女性每天 **1500ml**（7-8杯）
- 首选白开水，不喝或少喝含糖饮料
- 在高温环境或进行身体活动时，应适当增加饮水量

### 准则七：会烹会选，会看标签
- 选择新鲜的、营养素密度高的食物
- 学会阅读食品标签，识别营养成分
- 合理烹调，减少营养素损失

### 准则八：公筷分餐，杜绝浪费
- 选择新鲜卫生的食物和适宜的烹调方式
- 使用公筷公勺，分餐制
- 珍惜食物，按需备餐，合理储存

## 二、中国居民平衡膳食宝塔（2022）

### 每日推荐摄入量

| 层级 | 食物类别 | 每日推荐量 | 关键说明 |
|------|----------|------------|----------|
| 第一层 | 谷薯类 | 200-300g | 全谷物和杂豆50-150g，薯类50-100g |
| 第二层 | 蔬菜类 | 300-500g | 深色蔬菜占1/2 |
| 第二层 | 水果类 | 200-350g | ⚠️ 果汁不能代替 |
| 第三层 | 畜禽肉 | 40-75g | 优先禽肉 |
| 第三层 | 水产品 | 40-75g | 每周至少2次 |
| 第三层 | 蛋类 | 40-50g | **每天1个鸡蛋，不弃蛋黄** |
| 第四层 | 奶及奶制品 | 300-500g | 每天一杯奶 |
| 第四层 | 大豆及坚果 | 25-35g | 坚果10g左右 |
| 第五层 | 烹调油 | 25-30g | 优先植物油 |
| 第五层 | 食盐 | <5g | ⚠️ 2022版下调 |
| 饮水 | 白开水为主 | 男1700ml/女1500ml | 7-8杯，少量多次 |

## 三、蛋类摄入详细规则

### 3.1 基本原则

| 人群 | 每日推荐 | 每周上限 | 特别说明 |
|------|----------|----------|----------|
| 健康成年人 | 1个（40-50g） | 7个 | 不弃蛋黄 |
| 高血脂患者 | 半个蛋黄 | 3-4个 | 控制胆固醇 |
| 高胆固醇血症 | 不超过1个 | 5个 | 胆固醇<200mg/天 |
| 孕产妇 | 1-2个 | - | 可适当增加 |
| 体虚/术后恢复 | 2-3个 | - | 补充优质蛋白 |
| 儿童青少年 | 1个 | 7个 | 生长发育需要 |
| 健身人群 | 1个整蛋+额外蛋清 | - | 蛋清不限 |

### 3.2 蛋黄营养价值

**不建议丢弃蛋黄的原因：**
- 卵磷脂：大脑和神经系统发育
- 胆碱：记忆力和认知功能
- 叶黄素：眼睛健康
- 维生素A、D、E、K：脂溶性维生素
- B族维生素：能量代谢
- 锌、硒：免疫功能

### 3.3 胆固醇摄入规则

| 人群 | 每日胆固醇上限 | 1个鸡蛋胆固醇 | 剩余空间 |
|------|----------------|---------------|----------|
| 健康成年人 | 不设上限（合理即可） | ~200mg | 灵活 |
| 高血脂患者 | <200mg | ~200mg | ⚠️ 基本占满 |
| 心血管疾病 | <200mg | ~200mg | 需控制其他来源 |

⚠️ **高血脂患者注意**：1个鸡蛋≈200mg胆固醇，如果吃了1个鸡蛋，当天应避免其他高胆固醇食物（动物内脏、奶油、肥肉等）

## 四、蛋白质计算规则

### 4.1 每日需求公式

```
健康成年人 = 体重(kg) × 0.8g
轻体力劳动 = 体重(kg) × 1.0g
中等体力   = 体重(kg) × 1.2g
运动人群   = 体重(kg) × 1.2-1.5g
减重人群   = 体重(kg) × 1.2g（防止肌肉流失）
老年人     = 体重(kg) × 1.0-1.2g
孕早期     = 体重(kg) × 1.0g
孕中晚期   = 体重(kg) × 1.0g + 15g
哺乳期     = 体重(kg) × 1.0g + 20g
```

### 4.2 食物蛋白质换算（每份约7g蛋白质）

| 食物 | 重量 | 蛋白质 | 备注 |
|------|------|--------|------|
| 鸡蛋 | 1个（50g） | 6.5g | 含蛋黄 |
| 瘦肉 | 35g | 7g | 猪/牛/羊 |
| 禽肉 | 40g | 7g | 鸡/鸭 |
| 鱼虾 | 50g | 7g | 去骨 |
| 豆腐 | 100g | 8g | 北豆腐 |
| 牛奶 | 200ml | 6-7g | 约一杯 |
| 酸奶 | 150g | 5g | 无糖 |
| 豆浆 | 300ml | 5g | 纯豆浆 |

### 4.3 一日蛋白质分配示例（60kg成年人）

**目标：60 × 1.0 = 60g**

| 餐次 | 食物搭配 | 蛋白质 |
|------|----------|--------|
| 早餐 | 1个鸡蛋(6.5g) + 250ml牛奶(8g) + 全麦面包(3g) | 17.5g |
| 午餐 | 75g鸡胸肉(18g) + 50g豆腐(4g) + 米饭蔬菜(3g) | 25g |
| 晚餐 | 75g鱼肉(13g) + 蔬菜豆类(4g) | 17g |
| **合计** | | **59.5g** ✓ |

⚠️ **注意**：鸡蛋只安排1个！

## 五、特殊人群调整

### 5.1 糖尿病患者

| 项目 | 调整建议 | 原因 |
|------|----------|------|
| 主食 | 150-200g/天，全谷物为主 | 控制血糖波动 |
| 全谷物 | 占主食1/3以上 | 低GI，缓慢升糖 |
| 水果 | 选低GI水果，150-200g/天 | 避免血糖快速升高 |
| 蛋白质 | 1.0-1.2g/kg体重 | 有助血糖稳定 |
| 鸡蛋 | 每天1个，正常吃 | 蛋白质优质 |
| 用餐顺序 | 先蔬菜→蛋白质→主食 | 减缓血糖上升 |

**低GI水果**：草莓、蓝莓、樱桃、柚子、苹果、梨  
**高GI水果（限制）**：西瓜、荔枝、龙眼、芒果、菠萝

### 5.2 高血压患者

| 项目 | 调整建议 | 原因 |
|------|----------|------|
| 食盐 | **<5g/天**（严格执行） | 2022新标准 |
| 钾摄入 | 增加至3500mg/天 | 钾钠平衡 |
| 加工食品 | 避免或限制 | 隐形盐多 |
| 腌制品 | 避免 | 钠含量极高 |
| DASH饮食 | 推荐 | 降压效果明确 |

**高钾食物推荐**：香蕉、土豆、菠菜、橙子、豆类、牛油果

### 5.3 高血脂患者

| 项目 | 调整建议 | 原因 |
|------|----------|------|
| 胆固醇 | **<200mg/天** | 2022版标准 |
| 鸡蛋 | 半个蛋黄/天，或每周3-4个 | 1个蛋≈200mg |
| 内脏 | 避免 | 胆固醇极高 |
| 饱和脂肪 | <总能量10% | 用不饱和脂肪替代 |
| 反式脂肪 | <2g/天 | 人造奶油、油炸食品 |
| 膳食纤维 | 25-30g/天 | 帮助降低胆固醇 |

⚠️ **关于鸡蛋**：高血脂患者如果当天吃了1个鸡蛋，应避免其他高胆固醇食物

### 5.4 减重人群

| 项目 | 调整建议 | 原因 |
|------|----------|------|
| 能量缺口 | 300-500kcal/天 | 温和减重，不反弹 |
| 蛋白质 | **1.2-1.5g/kg** | ⚠️ 防止肌肉流失 |
| 主食 | 减少但**≥150g** | ⚠️ 不能完全不吃 |
| 碳水类型 | 低GI为主 | 延缓血糖上升 |
| 蔬菜 | 500g以上 | 增加饱腹感 |
| 进食顺序 | 汤→菜→肉→饭 | 减少主食摄入 |

## 六、禁止的建议清单

以下建议 **绝对不能出现** 在 AI 输出中：

| ❌ 错误建议 | ✅ 正确说法 | 依据 |
|-------------|-------------|------|
| 每天2个鸡蛋补蛋白质 | 每天1个鸡蛋，额外蛋白质从肉/鱼/豆获取 | 膳食指南推荐40-50g/天 |
| 高血脂患者不能吃鸡蛋 | 可以吃，每周3-4个，或每天半个蛋黄 | 适量吃蛋黄有益 |
| 吃鸡蛋要丢蛋黄 | 不弃蛋黄，蛋黄营养丰富 | 准则四明确要求 |
| 不吃主食减肥 | 减少主食但每天至少150g，优先全谷物 | 主食是必需的 |
| 多喝牛奶补钙 | 每天300-500ml即可 | 过量无益 |
| 水果代替蔬菜 | 蔬菜300-500g，水果200-350g，不能互相替代 | 营养成分不同 |
| 喝果汁等于吃水果 | 果汁损失膳食纤维，不能替代水果 | 准则三明确说明 |
| 坚果多多益善 | 每天一小把（10-25g） | 热量高 |
| 少吃盐就是不吃盐 | 每天<5g，不是不吃 | 钠也是必需的 |
| 高血压要每天<3g盐 | 2022版标准是<5g（普通人也是<5g） | 已统一标准 |

## 八、AI 输出检查清单

### 8.1 数量约束（必须检查）

- [ ] 鸡蛋：每天 **≤1个**（健康成年人）
- [ ] 鸡蛋：高血脂患者 **每周≤4个** 或 **每天半个蛋黄**
- [ ] 牛奶：每天 **300-500ml**（不是"多喝"）
- [ ] 主食：每天 **≥150g**（减重也不能更低）
- [ ] 蔬菜：每天 **≥300g**
- [ ] 水果：每天 **200-350g**
- [ ] 食盐：每天 **<5g**（2022新标准）
- [ ] 坚果：每天 **≤25g**

### 8.2 禁止项检查

- [ ] 没有推荐"每天2个鸡蛋"
- [ ] 没有推荐"丢弃蛋黄"
- [ ] 没有推荐"不吃主食"
- [ ] 没有说"多喝牛奶"（应该是300-500ml）
- [ ] 没有说"果汁等于水果"
- [ ] 没有说高血压患者<3g盐（2022版是<5g）

### 8.3 特殊人群检查

- [ ] 高血脂患者：提到胆固醇**<200mg/天**
- [ ] 高血压患者：提到食盐**<5g/天**
- [ ] 糖尿病患者：提到低GI食物
- [ ] 减重人群：提到蛋白质要**充足**（防止肌肉流失）

### 8.4 表述规范

- [ ] 使用"膳食指南推荐"而不是"建议多吃/少吃"
- [ ] 给出具体数字（g/ml）而不是模糊的"适量"
- [ ] 引用来源："根据《中国居民膳食指南（2022）》..."

## 九、身体活动建议

### 9.1 基本要求

| 人群 | 运动频率 | 运动时长 | 日常步数 |
|------|----------|----------|----------|
| 成年人 | 每周≥5天 | 累计≥150分钟（中等强度） | 6000步/天 |
| 老年人 | 每周≥5天 | 累计≥150分钟 | 量力而行 |
| 儿童青少年 | 每天 | ≥60分钟（中高强度） | - |

### 9.2 运动强度说明

| 强度 | 特征 | 示例运动 |
|------|------|----------|
| 低强度 | 心率略升，可正常说话 | 散步、太极拳 |
| 中等强度 | 心率加快，说话稍喘 | 快走、骑车、游泳、广场舞 |
| 高强度 | 心率明显加快，说话困难 | 跑步、跳绳、球类运动 |

### 9.3 推荐运动组合

**每周运动计划（成年人）：**
- 有氧运动：≥150分钟（中等强度）或≥75分钟（高强度）
- 抗阻运动：2-3天（肌肉力量训练）
- 柔韧性训练：每天（拉伸、瑜伽）
- 日常活动：每天6000步

### 9.4 久坐提醒

- 每小时起来活动一次
- 久坐超过1小时，起来走动3-5分钟
- 工作间隙做简单拉伸
- 站立办公、走路开会

## 十、2022版 vs 2016版 主要变化

| 项目 | 2016版 | 2022版 | 变化说明 |
|------|--------|--------|----------|
| 食盐 | <6g/天 | **<5g/天** | 下调1g |
| 鸡蛋 | 无明确 | 40-50g/天（1个） | 明确量化 |
| 蛋黄 | 无特别说明 | **不弃蛋黄** | 强调蛋黄价值 |
| 准则数量 | 6条 | 8条 | 增加2条 |
| 饮水 | 1500-1700ml | 1500-1700ml | 不变 |
| 奶制品 | 300g | 300-500g | 上调 |
